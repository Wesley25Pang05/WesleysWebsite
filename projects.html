This was a project I made back in high school, where I wanted to challenge myself with a harder project. I then fell in love with the game of chess and decided, might as well try to make one. Wordle on Discord

When wordle started becoming popular, I decided to try and make wordle as my first project. I also then realized I didn't want to play this myself, so I introduced it into some discord servers for multiple players to play and help each other in a wordle game. To start, you add a bot to your server, run the code with the bot using the API Key, then use the commands, play wordle, guess "word"
